# Парсинг новостного сайта

Учебный проект по парсингу HTML-страницы с помощью библиотеки **BeautifulSoup**.

Цель — обработать новостную страницу архива сайта [iz.ru](https://iz.ru) и
сгруппировать новости по тематическим разделам (например, *Общество*, *Здоровье*, *Спорт* и т.д.).

Основной скрипт проекта — `parse_news.py`.

---

## Источник данных

В задании используется сохранённая копия страницы из веб-архива:

`https://web.archive.org/web/20230903112115/https://iz.ru/news`

Скрипт сам обращается к этому адресу, загружает HTML-страницу и дальше
работает уже с её содержимым как с обычным текстом.

---

## Что делает программа

1. Отправляет HTTP-запрос к архивной странице новостей (через библиотеку `requests`).
2. Проверяет, что страница корректно загрузилась (код ответа не 404 и не ошибка).
3. Разбирает HTML с помощью `BeautifulSoup`.
4. Находит на странице «карточки» новостей.
5. Для каждой новости извлекает:
   - **название** (заголовок новости);
   - **ссылку** на полный текст новости;
   - **раздел** (тематику), к которому относится новость.
6. Формирует итоговую структуру данных в виде словаря Python:

```python
{
    "Общество": [
        {"title": "Заголовок новости 1", "link": "https://iz.ru/..."},
        {"title": "Заголовок новости 2", "link": "https://iz.ru/..."},
        ...
    ],
    "Спорт": [
        {"title": "Заголовок новости N", "link": "https://iz.ru/..."}
    ],
    ...
}
Выводит полученный словарь в консоль (при желании его можно
дополнительно сохранить в JSON-файл).

Таким образом, ключами словаря являются названия тематических блоков,
а значениями — списки новостей в этих разделах.

Зависимости

Проект использует:

beautifulsoup4 — парсинг HTML;

requests — загрузка страницы по HTTP.

Они перечислены в файле requirements.txt.

Установка

Рекомендуется использовать виртуальное окружение.

# создать виртуальное окружение
python -m venv venv

# активировать окружение
# Windows:
venv\Scripts\activate
# Linux / macOS:
source venv/bin/activate

# установить зависимости
pip install -r requirements.txt

Запуск

После установки зависимостей запустите основной скрипт:

python parse_news.py
